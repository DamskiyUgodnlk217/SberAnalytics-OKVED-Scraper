# СБАР Скрапер (СберАналитика)

Скрипт для автоматизированного сбора данных юридических лиц по кодам ОКВЭД из сервиса **СБАР** (премиум-тариф).

**Собираемые данные:**
- Название компании
- Дата регистрации
- ФИО руководителя / учредителя
- Email (до 2 шт.)
- Телефон (до 2 шт.)
- Прибыль за последний доступный год

**Отдельные выгрузки:**
- Медицинские организации
- Страховые компании
- IT / веб-разработка / мобильные приложения / внедрение CRM

## Важное предупреждение
- Скрипт работает **только** с твоим личным премиум-аккаунтом СБАР.
- Массовый парсинг может нарушать условия сервиса. Используй на свой страх и риск.
- Перед запуском замени все селекторы под актуальный интерфейс СБАР (они могут меняться).
