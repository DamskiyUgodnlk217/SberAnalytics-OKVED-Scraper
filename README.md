# SberAnalytics-OKVED-Scraper
SberAnalytics OKVED Scraper — инструмент для быстрого сбора данных юридических лиц из СберАналитики (премиум). По кодам ОКВЭД выгружает название, дату регистрации, ФИО руководителя и учредителей, email и телефоны (до 2 шт.), а также прибыль за последний год. Отдельные выгрузки: медицина, страхование и IT-компании.
